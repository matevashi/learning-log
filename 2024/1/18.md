## 漫画でわかるGit～コマンド編～
第十一話　git stashでコミットしたくないファイルを一時退避！

いったん編集していたファイルを横に置いときたいときに使えるコマンド

- git stash save (スタッシュを保存する)
- git stash list (保存したリストを見る)
- git stash pop [スタッシュ名] (スタッシュの復活と削除を同時に行う)
